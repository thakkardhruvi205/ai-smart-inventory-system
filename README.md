# AI Smart Inventory System

AI Smart Inventory System is a web-based inventory management application that helps businesses track products, manage sales transactions, and analyze inventory data using a smart dashboard.

## Features
- Smart Dashboard with analytics
- Revenue, Orders, and Product statistics
- Inventory management system
- Add transaction functionality
- Barcode scanner integration
- Voice assistant support
- Light/Dark theme toggle
- Transaction history tracking

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: VS Code, GitHub
